# 全自动张维为教授表情包fetch bot
Automatic VV meme retrieval
![迎头痛击](yingtoutongji.png)

## Getting Started
- 安装requirements.txt
- 命令行打入`streamlit run web.py`
- 浏览器(自动)前往`http://localhost:8501`

## Data Update
- 新的表情包图片放入`vvs`文件夹
- 运行`data_update.py`
